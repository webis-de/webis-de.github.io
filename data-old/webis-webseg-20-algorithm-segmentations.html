Synopsis

The Webis-WebSeg-20-Algorithm-Segmentations dataset contains the segmentations of five segmentation algorithms, one ensemble, and one baseline algorithm for the pages of the Webis-WebSeg-20 dataset. If you use this dataset in your research, please cite it using this paper. [download]
