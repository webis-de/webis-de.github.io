Research Information System*

Update Lecturenotes
Select the courses in the lecturenotes repository which are to be updated. The service generates the lecturenotes slides, the slides browser thumbnails, and the HTML files as specified in bibliographies-webis and finally updates the Webis website. For the following domains the specified pages are recreated:
  • webis.de: courses map, lecturenotes
Update Website
The service generates the HTML files as specified in bibliographies-webis and updates the Webis website. For the following domains the specified pages are recreated:
  • webis.de: completed theses, data, events, lecturenotes, publications
  • leipzig.webis.de: funded projects, graduate seminar, people
  • weimar.webis.de: funded projects, graduate seminar, people
  • pan.webis.de: data, publications
  • touche.webis.de: data, publications
Upload PDF Files
Select and upload PDF files (lecturenotes slides, papers, posters, talk slides, theses) for the Webis website from your local computer to the cloudflare repository. Clicking a link below will open a corresponding upload page:
  • webis.de: completed theses, lecturenotes, publications
  • pan.webis.de: publications
  • touche.webis.de: publications
Edit Website Sources
Edit the HTML source for those webpages of the Webis website that are not specified in bibliographies-webis. Manually maintained webpages (via GitHub Pages):
  • webis.de: courses map, facilities, for students, research
  • leipzig.webis.de: awards, people, research, teaching
  • weimar.webis.de: awards
  • pan.webis.de: events, organization, shared tasks
  • touche.webis.de: events, organization, shared tasks
