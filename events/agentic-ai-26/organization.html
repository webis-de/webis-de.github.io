Submission

Participants address one or more of three sub-tasks (retrieval, simulation, solving) and submit their runs in the prescribed formats together with an agent behavior log; validation tools are provided. Tool access is via an established AI agent protocol (e.g., MCP or A2A). See the Task page for the full participation process and the data and tools.

The submission platform and detailed instructions are to be announced.
