We are happy to host the German leg of the Hugging Face ML Demo.cratization tour. This hands-on workshop will be held in English and will discuss the lifecycle of a machine learning model after it's been trained. Specifically, you will learn how to create Machine Learning apps and demos using the Streamlit and Gradio libraries.

Additionally, you will learn how to share these apps with the world to better communicate the results of your research. This workshop will be led by Omar Espejel who is a Machine Learning Developer Advocate at Hugging Face.

To follow along the live coding session you will need:

  • Join the event's Organization Page on the Hugging Face Hub. (You will be prompted to create an account if you don't already have one)
