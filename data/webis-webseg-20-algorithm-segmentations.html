Synopsis

The Webis-WebSeg-20-Algorithm-Segmentations dataset contains the segmentations of five segmentation algorithms, one ensemble, and one baseline algorithm for the pages of the Webis-WebSeg-20 dataset.

Access

Please refer to the publications for citing the dataset. If you want to link the dataset, please use the dataset permalink [doi].
