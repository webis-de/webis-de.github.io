Webis-CausalQA-22

  • Synopsis
  • Download

Synopsis

The Webis Causal Question Answering 2022 (Webis-CausalQA-22) corpus comprises 1.1M causal question-answer pairs collected from the public QA datasets.

Download

You can access the Webis Causal Question Answering 2022 from download.

After download you should unzip the archive. The resulting directory (440 MB after unzipping) contains 9 zip archives named after the QA datasets from which causal QA pairs were extracted and a README file, which contains instruction for creating the 10th data file. Each dataset file is stored as a jsonl file.
Each QA pair is represented as a dictionary with the keys such as for example "question", "answer", "context", etc. The original key names were preserved to reflect the respective QA dataset structure and may vary from sample to sample that allows to run QA systems developed for a particular dataset on its causal sample for comparing the system's effectiveness.
