Synopsis

The Webis-Exhibition-Questions-21 comprises 849 manually categorized questions (557 English, 292 German) from 63 participants combined with a detailed interaction log, which allows replaying each session (29 hours total) for in-depth analyses. If you use this dataset in your research, please cite it using this paper. [replay]
