Synopsis

Webis-WikiDebate-18 is a large-scale corpus for the argumentation model. The corpus is generated automatically based on the metadata in discussions, and then verified partly by an expert.

Access

We kindly ask you to cite the dataset as [bib]. If you want to link the dataset, please use the dataset permalink [doi].

  • Download the dataset from Zenodo or from the Internet Archive
  • Find the related metadata at Google.
